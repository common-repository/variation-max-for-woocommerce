== Description ==
Update the default number of variations allowed in WooCommerce. 

You no longer need to edit wp-config.php directly which reduces the chance of causing errors on your Wordpress site. 

Another added bonus is that the option is located intuitively in the
WooCommerce > Settings > Advanced > Variation Max tab so that users can easily and quicly locate the option.

This plugin filters WC_MAX_LINKED_VARIATIONS, the constant that holds the variation max value.

Ordered list:

1. Edit variation maximum easily
2. Less error prone than editing wp-config.php
3. Intuitively placed in WooCommerce > Settings > Advanced > Variation Max 

== Installation ==
1. After downloading the ZIP file,
2. Log in to the administrator panel.
3. Go to Plugins Add > New > Upload.
4. Click "Choose file" ("Browse") and select the downloaded zip file.

**For Mac Users**
Go to your Downloads folder and locate the folder with the plugin. Right-click on the folder and select Compress. Now you have a newly created .zip file which can be installed as described here.

5. Click "Install Now" button.
6. Click "Activate Plugin" button for activating the plugin.

== Frequently Asked Questions ==
= How do I use it =
1. Simply navigate to WooCommerce > Settings
2. Click the 'Advanced' tab
3. Click the 'Variation Max' field, then enter you're numerical value
4. Click 'Save Changes', then navigate to your variations panel in WooCommerce
